# Experiments on program generation from problem reformulation in programming contexts with generative machines.


Each repository represents an experiment. A repository is a Java project that can be cloned from Eclipse or Intellij.

The project structure contains 3 main directories:

input: it contains the problem (problem-program.txt) and its reformulation (problem-program-reformulation.txt)

output: it contains the reasoning generated by the generative machine from the problem (reasoning.txt) and the reasoning generated by the generative machine from the problem reformulation (reasoning-reformulation.txt)
